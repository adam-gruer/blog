This a short post demonstrating how to link to another post from the .Rmd source of a separate post.

The directory of blog project looks a bit like this on my computer. Each .Rmd source file in your content/post folder has some YAML markup that will look something like this

The slug is the important thing for links. The slug in the YAML of my file 2018-06-01-guards-guards-purrr-map-if.Rmd is guards-guards-purrr-map-if . If I ant to link to it from this post I can use a relative link of /post/slug, so in this instance <a href=“/post/guards-guards-purrr-map-if”></a> Guards! Guards!
